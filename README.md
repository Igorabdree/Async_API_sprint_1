Асинхронный REST API для онлайн-кинотеатра

О проекте

Сервис представляет собой высокопроизводительное асинхронное REST API

Ключевые функции:

Получение детальной информации о фильмах из Elasticsearch

Кэширование данных в Redis для ускорения ответов и разгрузки основного хранилища

Расширенные возможности поиска с поддержкой фильтрации, сортировки и постраничной навигации

Технологический стек
Python 3.9+

FastAPI (асинхронный веб-фреймворк)

ElasticSearch (поисковый движок)

Redis (кэширование)

Docker & Docker Compose (контейнеризация)

Быстрый старт
Для развертывания приложения выполните следующие шаги:

Клонирование репозитория
$ git clone [URL репозитория]

Создание виртуального окружения
$ python -m venv venv

Активация виртуального окружения
$ source venv/bin/activate  # для Linux/MacOS
$ venv\Scripts\activate     # для Windows

Установка зависимостей
$ pip install -r requirements.txt

Запуск через Docker Compose
$ docker-compose up --build

Документация API
После успешного запуска приложения доступна интерактивная документация:
Swagger UI: http://localhost/api/docs

GET    /api/v1/films           # Список фильмов с пагинацией
GET    /api/v1/films/{id}      # Детальная информация о фильме
GET    /api/v1/films/search    # Поиск фильмов по названию
GET    /api/v1/genres          # Список жанров
GET    /api/v1/genres/{id}     # Фильмы определенного жанра
